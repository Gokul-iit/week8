# Customer Engagement Heatmap Visualization

This repository contains a professional-grade Seaborn visualization that displays customer engagement patterns across different marketing channels and time periods.

## Project Overview

The visualization is a heatmap that shows customer engagement rates (as percentages) across various marketing channels over multiple quarters. This helps businesses understand which channels perform best and identify seasonal trends in customer engagement.

## Files Included

- `chart.py` - Python script that generates the Seaborn heatmap visualization
- `chart.png` - Generated heatmap image (512x512 pixels)
- `README.md` - This documentation file

## Visualization Details

The heatmap displays:
- **Channels**: Email, Social Media, Website, Mobile App, SMS, Direct Mail
- **Time Periods**: Quarterly data from Q1 2023 to Q4 2024
- **Metrics**: Engagement rates as percentages with color coding
- **Color Scale**: Red-Yellow-Green palette where green indicates higher engagement

## Technical Specifications

- Uses Seaborn's `sns.heatmap()` function
- Professional styling with `sns.set_style("whitegrid")`
- Figure size: 8x8 inches for 512x512 pixel output
- DPI: 64 for exact pixel dimensions
- Color palette: RdYlGn (Red-Yellow-Green)
